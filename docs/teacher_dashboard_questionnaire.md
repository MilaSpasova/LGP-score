# Teacher Dashboard Questionnaire

The final questionnaire uses plain language and avoids technical terminology.

## Response Scale

`1` Strongly disagree  
`2` Disagree  
`3` Not sure  
`4` Agree  
`5` Strongly agree

## Closed Questions

- `Q1` The instructions were easy to follow.
- `Q2` It was easy to compare the rewritten versions.
- `Q3` It was easy to explain why I preferred one version over the others.
- `Q4` It was easy to point out words or phrases that felt odd or unclear.
- `Q5` Overall, the dashboard was easy to use.
- `Q6` I could imagine using a tool like this when reviewing texts for students.

## Open Questions

- `O1` What helped you decide which rewritten version was best?
- `O2` Was there anything confusing or difficult about the process?
- `O3` Is there anything you would change or add before using a tool like this again?
