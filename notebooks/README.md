# Notebooks

These notebooks are exploratory or reporting-oriented helpers. They are useful for inspection, appendix preparation, and result interpretation, but the thesis pipeline does not depend on them.

Current key notebooks:

- `results_full_explainer.ipynb`: detailed narrative walkthrough of the current benchmark and prompt-pilot results.
- `appendix_export.ipynb`: writes rounded appendix tables and copies selected thesis figures into `outputs/appendix_export`.
- `llm_results_visualization.ipynb`: visual reporting notebook for prompt-condition and LLM output comparisons.
- `results_analysis.ipynb`: CTAP-versus-local comparison notebook for sanity checking the fallback pipeline.
